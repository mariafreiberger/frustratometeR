# frustratometeR
An R package for caculating energetic local frustration

# installation 
`library(devtools)`

`install_github("gonzaparra/frustratometeR")`

# dependencies

**R packages**
bio3D
`install.packages("bio3d")`

**Python**
numpy 
`pip install numpy`
note: replace pip by pip3 if needed


**others**
pymol
`sudo apt install pymol`

perl

modeller
